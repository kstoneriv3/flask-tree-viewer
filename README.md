# Flask-Tree-Viewer

## Project Structure

```
flask-tree-viewer/
├── app.py
├── templates/
│   └── dashboard.html
└── static/
    ├── css/
    │   └── ...
    └── js/
        └── ...
```


## How to Install

```bash
$ pip install -e .
```


## How to Run

To host the app at localhost:5000, run
```bash
$ python app.py debug  # for debuggign
$ python app.py serve # for production
```
